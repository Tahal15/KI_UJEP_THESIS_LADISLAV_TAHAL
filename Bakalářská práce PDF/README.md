# Bakalářská práce PDF

Tento adresář obsahuje zdrojové kódy a výsledné PDF bakalářské práce ve formátu LaTeX.

## Struktura adresáře

*   `README.md`: Původní úvodní soubor k tomuto adresáři.
*   `AI_README.md`: Tento soubor, generovaný s pomocí AI pro lepší přehlednost.
*   `ki-thesis.tex`: Hlavní LaTeX soubor pro bakalářskou práci.
*   `thesis.bib`: BibTeX soubor s bibliografickými citacemi.
*   `img/`: Adresář obsahující obrázky použité v práci.
*   `*.pdf`: Výsledné PDF soubory bakalářské práce.
*   Ostatní soubory jsou pomocné soubory LaTeXu.
