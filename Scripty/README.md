# Scripty

Tento adresář obsahuje veškeré pomocné skripty pro ETL procesy, databázové operace a další.

## Struktura adresáře

*   `README.md`: Původní úvodní soubor k tomuto adresáři.
*   `AI_README.md`: Tento soubor, generovaný s pomocí AI pro lepší přehlednost.
*   `Python/`: Adresář s Python skripty.
    *   `ETL/`: Skripty pro ETL procesy pro různé databázové platformy.
    *   `MariaDB to MSSQL conversion/`: Skript pro konverzi dat z MariaDB do MSSQL.
    *   `Uvozovky/`: Skripty pro práci s uvozovkami v textech.
*   `SQL/`: Adresář s SQL skripty.
    *   `ClickHouse_DWH/`: Skripty pro ClickHouse datový sklad.
    *   `DWH/`: Skripty pro MSSQL datový sklad.
    *   `Maria_DWH/`: Skripty pro MariaDB datový sklad.
    *   `PostgreSQLTDB_DWH/`: Skripty pro PostgreSQL/TimescaleDB datový sklad.
*   `pgloader/`: Konfigurační soubory pro pgloader.
*   `Start sluzeb/`: Batch skripty pro spouštění databázových služeb.
